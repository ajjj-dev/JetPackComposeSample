
JetPackComposeSample                                           
A simple Android app demonstrating **Jetpack Compose** for building modern UI using Kotlin.

Features
* Built with Jetpack Compose
* Declarative UI approach
* Clean and simple project setup

 Tech Stack
* Kotlin
* Jetpack Compose
* Gradle (KTS)

Getting Started

'''' 
bash
git clone https://github.com/ajjj-dev/JetPackComposeSample.git
```

Open in Android Studio and run the app 

Structure
```
app/src/main/        # App code
app/src/test/        # Unit tests
app/src/androidTest/ # UI tests
```
